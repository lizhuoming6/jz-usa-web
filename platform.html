Core Features

Built for foam, not for generic IoT

01

Shot-by-Shot Logging

Every injection cycle captured: A/B temperature, pump pressure, gun pressure, flow rate, A/B ratio, cycle time — all timestamped and stored locally.

02

SPC & Process Control

Real-time Cp/Cpk calculations, X-bar/R control charts, and all 8 Western Electric rules — flagged automatically, no manual review needed.

03

Auto-Diagnosis

Pressure differential analysis across pump → gun path detects blockages, valve issues, and sensor drift before they cause defects.

04

Premix Management

16-ingredient batch weighing with recipe library, lot traceability, and supplier tracking — links raw material quality to finished foam results.

05

Batch Traceability

Full chain from raw material lot → premix batch → production shots → quality results. One-click export for customer audits.

06

Offline-First

Runs entirely on a local industrial PC. No internet required. Cloud backup is optional. Designed for factory floors where connectivity is unreliable.
